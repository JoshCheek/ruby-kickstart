There are two ways to represent a literal String in Ruby: with double quotes ( “ ” ) or single quotes ( ‘ ’ ). With double quotes, you can place special sequences, like newlines ( n ) in the string that will be converted. With single quotes, the string comes out just as it’s typed.

puts "Here's\nTwo lines."
puts 'This will\nonly be one line'

 # Output:
 # Here's
 # Two lines.
 # This will\nonly be one line
