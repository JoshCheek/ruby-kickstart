Well that seems pretty useless, eh? Fortunately, there is an internet connection on this world! You can pass objects, into the method. When we pass an object in, the variable that points to it is called an argument or a parameter

person1 and person2 are arguments or parameters

def beam_me_up( person1, person2 )
 "The away team consists of #{ person1 } and #{ person2 }."
end

beam_me_up( "Ryker", "Data" ) # => "The away team consists of Ryker and Data."
